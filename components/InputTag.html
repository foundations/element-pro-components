# InputTag

基于 tag input autocomplete 的输入多个标签的输入框

# 使用

  1. 基于 input 使用

input 默认 type="text"，支持 type="text" 的大部分配置

<template>
  <pro-input-tag
    v-model="inputTags"
    placeholder="请输入内容,点击空格按键"
    @change="handleTagsChange"
  />
</template>

<script>
export default {
  data() {
    return {
      inputTags: []
    }
  },
  methods: {
    handleTagsChange(tags) {
      console.log(tags)
    }
  }
}
</script>

# 配置

asideWidth

  • type: String
  • default: 300px

侧边宽度

headerHeight

  • type: String
  • default: 60px

顶栏高度

showFooter

  • type: Boolean
  • default: true

是否显示底栏

footerHeight

  • type: String
  • default: 60px

底栏高度

其它参考 Menu

# 插槽

  • asideTop
  • asideBottom
  • headerLeft 会替换默认 ProBreadcrumb
  • headerRight
  • footer 会替换默认
